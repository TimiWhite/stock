# 试盘线战法要领（结构化版）

## 1. 战法定位

- 试盘线是主力建仓后测试抛压和跟风盘的行为信号，可视为潜在拉升前兆。
- 核心不是“猜顶底”，而是利用规则化信号提高盈亏比与执行一致性。

## 2. 试盘线定义（必须同时满足）

1. 当日成交量 >= 前一日成交量 3 倍。
2. 股价处于相对低位，前期经历约 1-3 个月震荡整理。
3. 当日为真阳线，优先选择同时突破整理区间上沿的个股。

补充过滤：
- 优先横盘震荡或低点逐步抬高结构。
- 尽量排除下降趋势个股。

## 3. 生命线与买点

- 生命线定义：试盘线当日收盘价。
- 买点定义：自试盘线次日起，K 线实体向上突破生命线。
- 执行细节：优先在尾盘（约 14:45 后）确认突破有效性，减少假突破风险。

## 4. 失效与风控

- 止损：
  - 主板：生命线下方 3%-5%
  - 创业板/科创板：生命线下方 5%-8%
- 失效判定：突破后 5 个交易日内无大阳线/涨停，视作主力拉升意愿不足，降级观察。

## 5. 止盈信号（短线）

在已有一定涨幅（常见 20%-30%）后，以下信号可靠性更高：

1. 出现放巨量阴线。
2. 出现放巨量十字星或明显长上影线（长下影线在高位也需警惕）。
3. 连续 2-3 天长上影线。
4. 巨量达到试盘线量能约 3 倍或以上。

## 6. 形态辅助

- 常见整理形态：箱体、三角形、旗形。
- 形态突破与生命线突破共振时，主升浪启动概率更高。

## 7. 通达信选股公式（文档原式）

```text
倍量:=V/REF(V,1)>=2.9;
N:=REF(BARSLAST(倍量),1);
TJ1:=倍量;
TJ2:=C>REF(C,1)&&H>C&&H=HHV(H,6);
TJ3:=REF(LLV(L,10),1)>=REF(HHV(H,10),1)*0.8;
TJ4:=LLV(L,5)>LLV(L,20);
TJ5:=COUNT(倍量,60)>1;
TJ6:=V>REF(V,N);
TJ7:=C>REF(C,N)&&C>REF(O,N);
SPK:=TJ1&&TJ2&&TJ3&&TJ4;
XG:SPK;
```

说明：
- 原文最终信号为 `SPK:=TJ1&&TJ2&&TJ3&&TJ4`，`TJ5/TJ6/TJ7` 仅定义未参与最终输出。
- 若后续回测需要，可建立“基础版（SPK）”与“增强版（加入TJ5-7）”双版本对比。

## 8. 执行纪律

- 先设止损止盈，再入场。
- 触发止损立即执行，不拖延不摊平。
- 不追求卖在最高点，优先锁定可重复收益。
